## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps:
1. Click **"Code"** button
2. Copy [**clone link**](https://github.com/Botlhale-Setou/my-awesome-books.git)
3. Open **Git Bash** or other terminal.
4. Navigate to desired projects folder.
5. Enter 
	```javascript
	git clone https://github.com/Botlhale-Setou/my-awesome-books.git
	```

6. Enter 
	```javascript
	cd my-awesome-books
	```

7. Enter 
	```javascript
	start index.html
	```
